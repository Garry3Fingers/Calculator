  • Max input is 18 characters
  • You can dynamically change math operations

Keyboard Control:

  • Key 0-9
  • Shift + 8 - multiplication
  • Shift + = - addition
  • Key '/' - division
  • = - equal
  • - is subtraction
  • Shift + 9 - left parenthesis
  • Shift + 0 - right parenthesis
  • Key 'd' - undo last character
  • Key 'c' - clear everything
  • Key '.' - period
